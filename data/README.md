# Data

Local data storage, split into two zones per the System Architecture document:
- raw/ — exactly what was ingested, untouched. Kept for audit and reprocessing.
- validated/ — only data that has passed Phase 4 validation and bias checks. This is what Phase 5+ reads from.

This folder is excluded from version control (see .gitignore) — data is regenerated by running the ingestion and validation pipelines, not committed to git.
